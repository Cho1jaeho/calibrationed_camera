# calibrationed_camera
calibrationed camera program with OpenCV in python
<h1>소개</h1>
<p>체스보드를 동영상으로 촬영한 파일에 캘리브레이션을 추가해 보았다.</p>
<h1>기능</h1>
<ul>
  <li>1. 카메라에 캘리브레이션 수행</li>
  <li>2. 캘리브레이션 결과를 바탕으로 렌즈 왜곡 보정 수행</li>
</ul>
